# mysql-labs

MySQL labs

## Build

Ubuntu 20.04

``` bash
./build.sh build
```

this will:

* download and extract boost-1.73.0 to `/data/opt/boost/boost_1_73_0`
* build mysql-server to `dist/mysql-server`
